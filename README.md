# formulaIT_Python task 5.6
Необходимо написать функцию, в которой в бесконечном цикле генерируется 2 случайных натуральных числа (от 1 до 100) и в рамках математической игры запрашивается их сумма у пользователя. Если ответ пользователя неверный, то ему назначается штрафная пауза - он отстраняется от игры на какое-то количество секунд (используйте метод sleep). Иначе (при правильном ответе) цикл завершается. Пусть изначально штрафная пауза составит 3 секунды, но с каждым неправильным ответом она будет увеличиваться на 1 секунду.

Пример работы программы:

Программа выводит: 

Новый пример: 55 + 34?

Пользователь вводит с клавиатуры: 

47

Программа выводит: 

Неверно! Штрафная пауза: 3 секунды

Новый пример: 62 + 98?

Пользователь вводит с клавиатуры: 

150

Программа выводит: 

Неверно! Штрафная пауза: 4 секунды

Новый пример: 70 + 78?

Пользователь вводит с клавиатуры: 

148

Программа выводит: 

Поздравляю! Ваш ответ правильный
